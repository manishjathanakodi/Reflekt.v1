REFKotlin Implementation Quick Reference
DARK — DarkColorScheme
background = Color(0xFF0E1117)
surface = Color(0xFF161B27)
surfaceVariant = Color(0xFF1E2538)
primary = Color(0xFFC9A96E)
onBackground = Color(0xFFEEEAE2)
LIGHT — uses darkColorScheme() base
background = Color(0xFFF0EBE1) ← changes
onBackground = Color(0xFF2C2318) ← changes
surface = Color(0xFF1A2030) ← stays dark
surfaceVariant = Color(0xFF1E2538) ← stays dark
primary = Color(0xFFC9A96E) ← unchanged
THEME SWITCH — Theme.kt
val isDark = when(pref) {
  DARK → true
  LIGHT → false
  SYSTEM → isSystemInDarkTheme()
}
MaterialTheme(
  colorScheme = if(isDark) DarkCS else LightCS
)
NEVER CHANGES IN LIGHT MODE
Card backgrounds (#1A2030)
Input fields / TextFields (#1E2538)
AI chat bubbles (surfaceVariant)
User chat bubbles (primary gold)
Settings item backgrounds
Bottom sheet surfaces
Habit card backgrounds
All text inside cards (stays light)
